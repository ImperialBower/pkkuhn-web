# pkkuhn-web

Interactive Kuhn poker trainer powered by [pkcore](https://github.com/ImperialBower/pkcore) + WebAssembly.

## Kuhn Poker Resources

- [Kuhn Poker — Wikipedia](https://en.wikipedia.org/wiki/Kuhn_poker) — Overview of the game, rules, and Nash equilibrium analysis
- [A Simplified Two-Person Poker (Kuhn, 1951)](https://sites.math.rutgers.edu/~zeilberg/akherim/PokerPapers/Kuhn1951.pdf) — The original paper by H. W. Kuhn
- [Kuhn Poker: How a 3-Card Game Reveals the Art of Bluffing](https://medium.com/@parkt027/demystifying-the-simple-yet-complex-game-of-kuhn-poker-295f2c519d37) — Accessible introduction via Medium
- [Kuhn Poker with gtree](https://skranz.github.io/gtree/articles/t03_kuhn_poker.html) — Game-tree analysis walkthrough in R
- [Analysis of Kuhn Poker](http://www.eprisner.de/MAT109/AnalysisKUHNPOKER.html) — Detailed strategic analysis from a game theory course
- [3-Player Kuhn Poker](https://medium.com/ganzfried-gleans/3-player-kuhn-poker-project-ce1b818b4d5e) — Extension to three players by Sam Ganzfried
- [Building a Poker AI Part 6: Beating Kuhn Poker with CFR Using Python](https://ai.plainenglish.io/building-a-poker-ai-part-6-beating-kuhn-poker-with-cfr-using-python-1b4172a6ab2d) — Step-by-step CFR implementation in Python
- [Counterfactual Regret Minimization on Kuhn Poker — labml.ai](https://nn.labml.ai/cfr/kuhn/index.html) — Annotated CFR implementation with inline explanations
